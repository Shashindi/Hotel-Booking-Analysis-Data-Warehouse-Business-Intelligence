# Hotel-Booking-Analysis-Data-Warehouse-Business-Intelligence
End-to-end Data Warehousing &amp; BI solution for hotel review analytics — star schema DW, SSIS ETL with an accumulating snapshot fact table, an SSAS OLAP cube (MDX), Excel-based OLAP demos, and Power BI reports (matrix, cascading slicers, drill-down, drill-through)
